TelemetryNetSockets

Declaration

type TTelemetryNetServerClientWinSocket = class(TServerClientWinSocket)

Description

Descendant of TServerClientWinSocket used in server as client socket.

It implements incoming and outgoing buffers and set of state flags used in communication. Each client connecting to the server spawns instance of this class and this object is then used for server-client communication and client management.

Object used to buffer incoming data and process them into usable packets. Use this object event OnPacket to actually receive packets.

property UniqueIdentificator: TGUID read fUniqueIdentificator;

Unique identificator of client connection. It is automatically generated in constructor.

property WaitingForPassword: Boolean read fWaitingForPassword write SetWaitingForPassword;

Value of True means the client is connected but is not logged to the server and must send login packet (password) - when different packet is received, it is assumed the client does not know right password and is immediately disconnected. False means it has logged successfuly.
When set to different value then it already has, OnStatusChange event is called.

property ReadyToWork: Boolean read fReadyToWork write SetReadyToWork;

True means the client is ready to fully communicate with server. It is set to True only after client sends ready packet - meaning client decides whether it is ready, not the server.
When set to different value then it already has, OnStatusChange event is called.

property AdminRights: Boolean read fAdminRights write SetAdminRights;

Indicates wheather client has administrative rights.
When set to different value then it already has, OnStatusChange event is called.

property SuperAdminRights: Boolean read fSuperAdminRights write SetSuperAdminRights;

Indicates wheather client has super-administrator rights.
When set to different value then it already has, OnStatusChange event is called.

property OnStatusChange: TNotifyEvent read fOnStatusChange write fOnStatusChange;

Event called whenever status of this socket changes (properties WaitingForPassword, ReadyToWork, AdminRights and SuperAdminRights).
