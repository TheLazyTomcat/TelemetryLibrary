Unit TelemetryNetPacketsResolving

DescriptionUsesClasses, Interfaces, Objects and RecordsFunctions and ProceduresTypesConstantsVariables

Description

This unit provides functions for packet reading and processing.

*******************************************************************************

TelemetryNetPacketsResolving

©František Milt, all rights reserved.

This unit is designated to provide functions that simplifies packets reading and processing.
There are three function groups and few other (not grouped) functions.
The groups are:

  • ReusePacket functions - Functions intended to provide a way to reuse allready created packets (reducimg memory allocation). For example, you can change ID of just received packet and send it back as response.

  • Ptr_Readout* functions - Functions designed to provide simple way of reading variables from general memory location given by pointer. Read value is returned as function result.

  • Payload_Readout* functions - These functions read value from payload of given packet. They all have paramter "Offset", which tells the function offset in bytes from the beginning of packet payload at which function should start reading. When the offset is not specified, function reads right at the beginning of payload.

Function MinimalPacketPayloadSize(PacketID: LongWord): Integer;

Returns minimal packet payload size.

Each packets payload must have minimal size given by its structure. This function returns this size for each packet type - if unknown packet type ID is passed, an exception is raised.
This function should be used to check received packet before further processing.
For actual values, refer to function implementation.

Parameters
PacketID
ID of packet type for ehich the minimum size is required.
Returns

Minimal size of payload for given packet ID.

Function PacketEventConfigurationToStr(const Data: Pointer; TypeName: Boolean = False; ShowDescriptors: Boolean = False): String;

Returns textual representation of data in TN_PACKET_EVENT_EVENT packet for SCS_TELEMETRY_EVENT_configuration event.

It is intended as replacement for function TelemetryRecipientAux.TelemetryEventConfigurationToStr, because structure of payload in mentioned packet differs form structure passed by configuration event.

Parameters
Data
Pointer to packet payload.
Type
name When set, value type identifiers for individual attribute values are added to output.
ShowDescriptors
When set, fileds descriptors are shown in component values.
Returns

Textual representation of given packet payload.
