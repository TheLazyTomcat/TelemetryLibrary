Function GetItemID(Item: String): TItemID;

Function to get identificator of passed item name. At the moment, identificators are implemented as CRC32 checksum of ansi version of given string.

Parameters
Item
Item name.
Returns

Item identificator.

Function GetMasterID(ID: TChannelID; Index: scs_u32_t; ValueType: scs_value_type_t): TMasterID; register;

Function used to calculate master identification number which is used in TStoredChannelValuesList.


This number is used in searching and sorting algorithms and is calculated from channel ID, Index and ValueType.
At the moment, it is calculated by the formula:

MasterID := (ID xor Index) xor not ValueType;

Parameters
ID
Channel ID.
Index
Channel index.
ValueType
Channel ValueType.
Returns

Master identification number.
