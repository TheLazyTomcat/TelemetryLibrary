Types

TAddressListMode = (...);

Used in TAddressList to set how the list will distinguish whether client can connect or not.

Values
  • lmBlackList: When client is listed, it cannot connect.
  • lmWhiteList: Client can connect only when listed.
  • lmAllowAll: All clients can connect.
  • lmDenyAll: No client can connect (use with caution).
