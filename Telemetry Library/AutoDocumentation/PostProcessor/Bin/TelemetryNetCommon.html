Constants

def_ServerAddress = '127.0.0.1';

Default (can be changed to user defined value) address at which the server is expected to run.

def_ServerPort = 8753;

Default (can be changed to user defined value) port at which the telemetry server will listen for incoming connections.

cLargeBufferThreshold = 4096;

Maximum amount of data small buffer can accommodate.

This value is used when reading data from a socket. If amount of data received is larger than this value, then large buffer must be allocated, if it is lower or equal, then small on-stack buffer can be used. This behavior is there to avoid unnecessary memory allocation for small amount of data.

cEmptyGUID: TGUID = '{00000000-0000-0000-0000-000000000000}';

Empty (zeroed-out) TGUID structure. It is used as placeholder when actual GUID is not known or needed.
