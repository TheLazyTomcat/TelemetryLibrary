Class TRegisteredEventsList

Unit

TelemetryLists

Declaration

type TRegisteredEventsList = class(TCustomTelemetryList)

Description

List used to store contexts of registered events.

When new event is registered in the telemetry API, it is registered with context which is actually pointer to a variable of TEventContext structure. This variable is at the same time added as a new item into this list.
When event is unregistered, context it is bound to is removed from this list.

Function CreateContext(Recipient: TObject; Event: scs_event_t; Utility: Boolean = False): PEventContext; virtual;

This method creates new event context variable, fills it from parameters and returns pointer to this variable. It is used when registering new telemetry event, because actual context is needed for this registration, but cannot be added to this list before the registration is complete.
Sequence of actions taken during registration is as follows:

  • Context is created using this function.

  • Registration in API is attempted.

  • When successful, context is added to this list.

  • When unsuccessful, context is freed using FreeContext method.

Telemetry recipient registering created context.

property Contexts[Index:Integer]: PEventContext read GetEventContext;

Array property mapped directly to internal list. Use it for direct access to individual stored contexts.
Unlike Events[] property, you can use returned pointer to change values of stored items.

property Events[Index:Integer]: TEventInfo read GetEventInfo;

Array property mapped directly to internal list. Use it to obtain values of individual stored items.
This property does not return whole context structure, only its "EventInfo" field.
