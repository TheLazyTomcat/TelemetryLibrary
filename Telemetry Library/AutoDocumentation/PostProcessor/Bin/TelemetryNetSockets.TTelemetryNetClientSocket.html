TelemetryNetSockets

Declaration

type TTelemetryNetClientSocket = class(TCustomSocket)

Description

Reimplementation of TClientSocket class.

This class is implemented in the same way as TClientSocket in Delphi 7, only differnce is, it uses TTelemetryNetClientWinSocket as endpoint socket instead of TClientWinSocket - because TTelemetryNetClientWinSocket implements buffers used during communication. There is currently no other method how to force use of TTelemetryNetClientWinSocket other than completely reimplementing the client socket.

For members description, refer to delphi help/documentation.
