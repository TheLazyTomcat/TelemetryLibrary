Class TTelemetryAbstractVersionObject

DescriptionHierarchyFieldsMethodsProperties

Unit

TelemetryVersionObject

Declaration

type TTelemetryAbstractVersionObject = class(TObject)

Description

These four methods: 

    HighestSupportedTelemetryVersion
    SupportsTelemetryVersion
    SupportsGameVersion
    SupportsTelemetryAndGameVersion

...can and actually must be called directly on class. They should be used to check whether the class supports required telemetry and game version before instantiation (creation of class instance).
