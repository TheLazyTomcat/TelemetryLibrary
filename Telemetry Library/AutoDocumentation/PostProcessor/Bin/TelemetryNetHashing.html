Function IsTextHash(Text: String; HashType: THashType; Hash: Pointer): Boolean;

Checks whether given hash is the same as hash of given string.
When used hash function is not known or any error occurs, function returns False.

Parameters
Text
String controled for hash.
HashType
Type of hashing function used to create hash.
Hash
Pointer to actual hash.
Returns

True when hash fits, otherwise False.
