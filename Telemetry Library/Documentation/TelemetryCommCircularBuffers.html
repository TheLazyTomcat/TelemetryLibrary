Description

Types

TCircularBufferItemNotifyEvent = procedure(Sender: TObject; Item: Pointer) of object;

Event type used in circular buffers on item operations (drop, addition, etc.). Item parameter contains pointer to data field of affected item.

TDefferedOperationType = (...);

Type used in TDefferedOperationsBuffer denoting what type of operation was deffered and has to be executed later.

Values
  • dotNone: No operation is deffered.
  • dotEventReg: Event registration is deffered.
  • dotEventUnreg: Event unregistration is deffered.
  • dotEventRegByIndex: Event registration by index is deffered.
  • dotEventUnregIndex: Event unregistration on index is deffered.
  • dotEventUnregByIndex: Event unregistration by index is deffered.
  • dotEventRegAll: Registration of all known events is deffered.
  • dotEventUnregAll: Unregistration of all registered events is deffered.
  • dotChannelReg: Channel registration is deffered.
  • dotChannelUnreg: Channel unregistration is deffered.
  • dotChannelRegAll: Registration of all known channels is deffered.
  • dotChannelUnregAll: Unregistration of all registered channels is deffered.
  • dotChannelRegByIndex: Channel registration by index is deffered.
  • dotChannelUnregIndex: Channel unregistration on index is deffered.
  • dotChannelUnregByIndex: Channel unregistration by index is deffered.
  • dotChannelRegByName: Channel registration by name is deffered.
  • dotChannelUnregByName: Channel unregistration by name is deffered.
