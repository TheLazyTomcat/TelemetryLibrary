Unit

TelemetryLogBinaryParser

Declaration

type TTelemetryLogBinaryFileParser = class(TTelemetryLogBinaryStreamParser)

Description

Class designed to parse a file containing binary log.

It is implemented as direct descendant of TTelemetryLogBinaryStreamParser class, you can refer to it for more details.
It creates TFileStream for requested file and passes it to inherited constructor.
