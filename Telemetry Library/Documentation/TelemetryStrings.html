Unit TelemetryStrings

DescriptionUsesClasses, Interfaces, Objects and RecordsFunctions and ProceduresTypesConstantsVariables

Description

Unit providing routines operating on TelemetryString type and routines converting selected binary types to text.

TelemetryStrings

©František Milt, all rights reserved.

This unit is intended to provide some basic routines for manipulation and processing of TelemetryString type (UTF8 encoded string) and also routines designed to return human readable (i.e. textual) representation of binary data stored in variables of selected types.

Function TelemetrySameStr(const S1, S2: TelemetryString): Boolean;

Compares strings based on the current locale with case sensitivity.

Since the TelemetryString is UTF8-encoded and there is no function for comparison of such strings, both strings are converted to WideString before actual comparison takes place.
This function can be slow, so if performance is important, consider using TelemetrySameStrNoConv instead.

Parameters
S1
First string to compare.
S2
Second string to compare.
Returns

True when the strings have the same value, False otherwise.

Function TelemetrySameText(const S1, S2: TelemetryString): Boolean;

Compares strings based on the current locale without case sensitivity.

Since the TelemetryString is UTF8-encoded and there is no function for comparison of such strings, both strings are converted to WideString before actual comparison takes place.
This function can be slow, so if performance is important, consider using TelemetrySameTextNoConv instead.

Parameters
S1
First string to compare.
S2
Second string to compare.
Returns

True when the strings have the same value, False otherwise.

Function TelemetrySameStrNoConv(const S1, S2: TelemetryString): Boolean;

Compares strings based on the current locale with case sensitivity and without internal conversions.

Unlike TelemetrySameStr, this function does not convert input strings to WideString before comparison. Instead, both strings are treated as normal AnsiString. This requires that both strings contains only ASCII characters (that is, up to #126), otherwise the function can, and probably will, return wrong result.

Parameters
S1
First string to compare.
S2
Second string to compare.
Returns

True when the strings have the same value, False otherwise.

Function TelemetrySameStrSwitch(const S1, S2: TelemetryString): Boolean;

Compares strings based on the current locale with case sensitivity.

This function internally calls TelemetrySameStrNoConv when switch AssumeASCIIString is defined. When it is not defined, it calls TelemetrySameStr.
