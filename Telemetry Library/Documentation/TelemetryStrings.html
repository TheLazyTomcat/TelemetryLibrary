Unit TelemetryStrings

DescriptionUsesClasses, Interfaces, Objects and RecordsFunctions and ProceduresTypesConstantsVariables

Description

Provides routines operating on TelemetryString type and routines converting selected binary types to text.

TelemetryStrings

©2013-2016 František Milt, all rights reserved.

Last change: 2016-03-20

This unit is intended to provide some basic routines for manipulation and processing of TelemetryString type (UTF8 encoded string) and also routines designed to return human readable (ie. textual) representation of binary data stored in variables of selected types.

Uses

  • TelemetryCommon

Function TelemetrySameStr(const S1, S2: TelemetryString): Boolean;

Compares strings based on the current locale with case sensitivity.

Implementation and behavior of this function depends on used compiler and whether symbol Unicode is defined. Currently it is as follows: 

    Delphi (Unicode)
        Strings are converted to UTF16 and compared using function AnsiSameStr

    Delphi (non-Unicode)
        Strings are converted to UTF16 and compared using function WideSameStr

    FPC (Unicode)
        Same as Delphi (Unicode)

    FPC (non-Unicode)
        Strings are NOT converted and are compared using function AnsiSameStr

Parameters
S1
First string to compare.
S2
Second string to compare.
Returns

True when the strings have the same value, False otherwise.
