Declaration

type TTelemetryLogBinaryToTextConverter = class(TTelemetryLogBinaryFileParser)

Description

Converts given binary log file to a text log file.

Implemented as descendant of TTelemetryLogBinaryFileParser class. It creates internal text logger and uses own methods to pass parsed data to it. It also creates dummy telemetry recipient using data from API information section because it is required by text logger.

constructor Create(FileName: String; OutFileName: String = '');

Class constructor.

Creates dummy recipient and text logger.

Parameters
FileName
Name of the input log file. Must contain valid file name, but it is not mandatory to assign full file path.
OutFileName
Name of output text file. When left empty, then name of output file is created by appending .LOG extension to the name of the input file.
Exceptions raised
ETLUnsupportedGame
When telemetry version, game and/or its version are not supported by this class.
