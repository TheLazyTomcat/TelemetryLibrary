TelemetryLogBinaryParser

Declaration

type TTelemetryLogBinaryStreamParser = class(TObject)

Description

Class designed to parse a stream containing binary log.

It is designed as a stream parser, meaning it reads over the input stream and when it runs into a valid log, it reads it, processes it when possible and calls appropriate event to which it passes all processed data. For example, when it finds entry containing information about channel registration, it reads it and calls event OnChannelRegisterLog (or similar alternative). These events have the same signature as events used by telemetry recipient, so you can bind the same handlers to them.
Parser internally uses appropriate readers to read files with different structures.

Reads next log entry from current position in Stream. Read data are processed and passed to appropriate event.
Calls internal readers ReadNextLogEntry method.

Returns

True when log was read successfuly, False otherwise.

procedure ReadAllLogEntries; virtual;

Reads all log entries from current position in Stream.
Calls internal readers ReadAllLogEntries method.

Function ReadLogEntry(Index: Integer): Boolean; virtual;

Reads log entry selected by passed index. Works only for structures that supports non-sequential reading (eg. structure 0).
Calls internal readers ReadLogEntry method.
Position in Stream is not changed in this method.

Parameters
Index
Index of requested log entry.
Returns

True when the entry was read successfuly, False otherwise.

Properties

property LogEntries[Index:Integer]: TLogEntry read GetLogEntry;

Array property mapped to log entries - that is, each item in this array corresponds to one log entry.
When you try to access item that is out of allowed boundary or the reading fails, then it returns cInvalidLogEntry.
Works only for structures that supports non-sequential reading.

Information about file that is being read. Filled in constructor.

property Stream: TStream read fStream;

Contains reference to a stream that is used to read the log. Not initialized, filled in constructor.
