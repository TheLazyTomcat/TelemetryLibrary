type scs_named_value_localized_t = record

Description

Structure used to store content of scs_named_value_t variable.

scs_named_value_t is using pointers for some values, so it cannot be stored directly, as only reference and no actual data would be stored.
Use this structure along with call to scs_named_value_localized function to store content of a variable of type scs_named_value_t.
