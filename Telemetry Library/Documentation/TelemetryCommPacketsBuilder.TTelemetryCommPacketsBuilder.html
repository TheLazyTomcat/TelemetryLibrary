Unit

TelemetryCommPacketsBuilder

Declaration

type TTelemetryCommPacketsBuilder = class(TTelemetryCommPacketsAllocator)

Description

Class providing methods that creates complete initialized predefined packets and few more methods intended for packets resolving.

There are two main function groups and few other (not grouped) functions. Main groups are:

  • BuildPacket functions - Set of functions providing ready-to-be-sent packet buffers. Use them to create packets with zero or one parameter (refer to individual functions for details).

  • BuildPacket_TC_PACKET_* functions - These functions are designed to provide complete, ready-to-be-sent, packets of specific packet ID. Each function identifier starts with BuildPacket_ followed by appropriate packet ID identifier (packet of this ID is returned by such function).
    Each function has such parameters that are required to build appropriate packet (paramerers for individual functions are not documented as they should be self-explanatory). Refer to packet definition or function implementation for details.

Function PreparePacket(var Packet: TPacketBuffer; PacketID: Integer): Pointer; virtual;

Method used for packet preparation.
It expects the field Size in Packet parameter to be set to size the packet will have (if it is set to value lower than is required for given packet ID, then an exception is raised). Data field of Packet parameter must not be initialized, since it will be initialized (allocated) in this method (to size passed in Packet.Size). Method also fills all information in packet header.

Parameters
Packet
Packet that has to be initialized in this routine.
PacketID
ID of initialized packet.
Returns

Pointer that points to a beginning of packet payload (when packet has no payload, it points to the end of the packet).

Function MinimalPacketPayloadSize(PacketID: LongWord): Integer; virtual;

Returns minimal packet payload size.
Each packets payload must have minimal size given by its structure. This method returns this size for given packet ID - if unknown packet ID is passed, an exception is raised.
This method should be used to check received packet before further processing.
For actual values, refer to implementation.

Parameters
PacketID
ID of packet for which the minimum size is required.
Returns

Minimal size of payload for given packet ID.

procedure ReusePacket(var Packet: TPacketBuffer; NewPacketID: LongWord; Address: Pointer; DataSize: Integer; Data: Pointer); overload; virtual;

Use to change packets ID and store new data to it.
Data are store at given address. When data would not fit into packet or address is outside of packet memory image, an exception is raised.

Parameters
Packet
Packet to be changed.
NewPacketID
New packet ID to be written to reused packet.
Address
Adress at which the data will be written.
DataSize
Size of data to be written.
Data
Data to be writen to packet.
