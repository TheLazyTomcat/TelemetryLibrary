Declaration

type TTelemetryRecipientBinder = class(TObject)

Description

Class designed as a base for any class that is supposed to bind itself to telemetry recipient events.

It has predefined abstract methods that can be assigned as handlers to recipient events one by one (when you want full control over what and when is called), or at once using AssignHandlers method (Recipient must be assigned for this to work).

Function GetWorkingRecipient(Sender: TObject; out Recipient: TTelemetryRecipient): Boolean; virtual;

Method used to get valid telemetry recipient for work.
When Recipient property is assigned, then it is returned in output parameter Recipient, when not, check whether object passed in parameter Sender is of type TTelemetryRecipient (or its descendant) and when it is, then this object is returned, otherwise output parameter contains Nil.

Parameters
Sender
Object suggested as working telemetry recipient.
Recipient
Output variable containing either valid telemetry recipient or Nil.
Returns

True when output parameter Recipient contains valid telemetry recipient, False otherwise.

constructor Create(Recipient: TTelemetryRecipient = nil);

Class constructor.

Parameters
Recipient
Telemetry recipient that operates on API. When you assign this parameter, it will be assigned to Recipient property and handlers will be assigned to its events using AssignHandlers method.
destructor Destroy; override;

Class destructor.

Calls method DeassignHandlers when DeassignHandlersOnDestroy property is True.

Function AssignHandlers: Boolean; virtual;

Assigns appropriate *Handler methods as handlers of Recipent events.
When symbol MulticastEvents is defined, the handlers are assigned to multicast events, otherwise they are assigned to normal events.

Returns

True when Recipient is assigned and all handlers were assigned to its event, False otherwise.

Function DeassignHandlers: Boolean; virtual;

Deassigns handlers of Recipient events.

Returns

True when Recipient is assigned and all handlers were deassigned, False otherwise.
