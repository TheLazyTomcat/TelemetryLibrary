This file is intended to provide types, constants, routines, etc. used throughout the Telemetry library (that is, in more than one unit). It also contains declaration of exception classes used in the library, those are: 

  ETLException
   |- ETLUnsupportedAPI
   |- ETLUnsupportedGame
   |- ETLIndexOfBounds
   |- ETLNilReference
   |- ETLInvalidReference
   |- ETLUnknownData
   |- ETLBufferTooSmall
   |- ETLRegFailed
   |- ETLInitFailed
   |- ETLBadData

Overview

Classes, Interfaces, Objects and Records

Name Description
Class ETLException Common ancestor for all exception classes in the library.
Class ETLUnsupportedAPI Raised when there is a need to operate on telemetry API but its version is not supported.
Class ETLUnsupportedGame Raised when there is need to operate on telemetry provided by a game that is not supported.
Class ETLIndexOfBounds Raised when passed index is out of allowed boundary (arrays, lists, ...).
Class ETLNilReference Raised when a nil reference (pointer or object) is passed to where a valid reference is required.
Class ETLInvalidReference Raised when passed reference is of wrong type or class, or is generally invalid.
Class ETLUnknownData Raised when code needs to work with some data but they are of unknown type or structure.
Class ETLBufferTooSmall Raised when a memory block or file is smaler than necessary.
Class ETLRegFailed Raised when registration of telemetry event or channel fails.
Class ETLInitFailed Raised when an initialization of some part of the code fails.
Class ETLBadData Raised when data the library works with are in some way corrupted or invalid.
Class ETLAlreadyExists Raised when item that is added to some list or array already exists there.
Class ETLNotfound Raised when requested item is not found in a list or array.
record TSupportedGame Structure used in lists of supported games and their versions.
record TDoubleIndex Used where there is need to pass or return two indices to fully define some reference (eg. indices in two-dimensional array).
record TConfigReference Structure used to reference a specific config (an attribute in a specified configuration).
record scs_value_localized_t Structure used to store content of scs_value_t variable.
record scs_named_value_localized_t Structure used to store content of scs_named_value_t variable.
record scs_telemetry_configuration_localized_t Structure used to store content of scs_telemetry_configuration_t variable.

Checks whether passed double index is valid (none of the field can be smaller than zero).
