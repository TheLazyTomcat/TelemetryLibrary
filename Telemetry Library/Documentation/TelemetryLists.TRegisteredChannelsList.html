Class TRegisteredChannelsList

DescriptionHierarchyFieldsMethodsProperties

Unit

TelemetryLists

Declaration

type TRegisteredChannelsList = class(TCustomTelemetryList)

Description

List used to store contexts of registered channels.

When new channel is registered in the telemetry API, it is registered with context which is actually pointer to a variable of TChannelContext structure. This variable is at the same time added as a new item into this list.
When channel is unregistered, context it is bound to is removed from this list.

Function CreateContext(Recipient: TObject; const Name: TelemetryString; Index: scs_u32_t; ValueType: scs_value_type_t; Flags: scs_u32_t; IndexConfig: TConfigReference; UserData: Pointer = nil): PChannelContext; virtual;

This method creates new channel context variable, fills it from parameters and returns pointer to this variable. It is used when registering new telemetry channel, because actual context is needed for this registration, but cannot be added to this list before the registration is complete.
Sequence of actions taken during registration is as follows:

  • Context is created using this function.

  • Registration in API is attempted.

  • When successful, context is added to this list.

  • When unsuccessful, context is freed using FreeContext method.

property Contexts[Index:Integer]: PChannelContext read GetChannelContext;

Array property mapped directly to internal list. Use it for direct access to individual stored contexts.
Unlike Channels property, you can use returned pointer to change values of stored items.

property Channels[Index:Integer]: TChannelInfo read GetChannelInfo;

Array property mapped directly to internal list. Use it to obtain values of individual stored items.
This property does not return whole context structure, only its "ChannelInfo" field.

property OnUserDataFree: TUserDataFreeEvent read fOnUserDataFree write fOnUserDataFree;

Bind this event when you need to free user data stored inside a context when such context is destroyed.
