TelemetryLogBinaryParser

Declaration

type TTelemetryLogBinaryReader_0_1 = class(TTelemetryLogBinaryReader)

Description

Internal reader class used to read file of structure 0 and 1.

Note - when reading payload af a block, the position in Stream advances exactly by a value stored in a block header as payload size, no more, no less.
