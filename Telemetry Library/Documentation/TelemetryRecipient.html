Description

Telemetry recipient class (API control and data receiver).

TelemetryRecipient

©2013-2016 František Milt, all rights reserved.

Last change: 2016-03-20

This unit contains TTelemetryRecipient class that used to control the telemetry API (see class declaration for details) and few classes used to manage multicast events for the recipient, namely: 

  TMulticastLogEvent
  TMulticastEventRegisterEvent
  TMulticastEventEvent
  TMulticastChannelRegisterEvent
  TMulticastChannelUnregisterEvent
  TMulticastChannelEvent
  TMulticastConfigEvent

Note that these are not part of the documentation.

Included files:

    .\Inc\TelemetryMulticastEvents.pas
      Contains declarations and implementations of multicast event classes.

Functions and Procedures

Function RecipientGetChannelIDFromName(const Name: TelemetryString; Recipient: Pointer): TChannelID;

Function intended as callback for streaming functions, converting channel name to ID.

UserData passed to streaming function along with this callback must contain valid TTelemetryRecipient object.

Parameters
Name
Channel name to be converted to ID.
Recipient
TTelemetryRecipient object that will be used for actual conversion.
Returns

Channel ID obtained from passed name.

Types

TGameCallback = (...);

Enumerated type used in method TTelemetryRecipient.SetGameCallback to set one particular game callback.

Values
  • gcbLog: Passed function pointer will be assigned to log callback.
  • gcbEventReg: Passed function pointer will be assigned to event registration callback.
  • gcbEventUnreg: Passed function pointer will be assigned to event unregistration callback.
  • gcbChannelReg: Passed function pointer will be assigned to channel registration callback.
  • gcbChannelUnreg: Passed function pointer will be assigned to channel unregistration callback.

Constants

MaxChannelIndex = 13;

Maximum allowed channel index.

This number is used when registering indexed channel where the channel does not have default maximum index value, value of the config containing count is not known or the channel is not binded to any config.
