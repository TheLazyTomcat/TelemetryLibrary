TelemetryLists

Declaration

type TStoredChannelsValuesList = class(TCustomTelemetryList)

Description

List used to store channels values obtained from API calls.

Items in this list are sorted by their master ID.

Function StoreChannelValue(const Name: TelemetryString; ID: TChannelID; Index: scs_u32_t; Value: p_scs_value_t): Integer; virtual;

Method used to store new value for already present channel. If such channel is not yet stored in the list, it is added as new item. Function can, in extreme cases, return -1 (when the channel is not already stored and addition fails for some reason).
OnChange event is called after successful addition or value change.
