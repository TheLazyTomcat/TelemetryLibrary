Properties

property Pointers[Index:Integer]: PStoredConfig read GetStoredConfigPointer;

Array property mapped directly to internal list. Use it for direct access to individual stored items.
Unlike Configs property, you can use returned pointer to change values of stored items.

property Configs[Index:Integer]: TStoredConfig read GetStoredConfig;

Array property mapped directly to internal list. Use it to obtain values of individual stored items.
