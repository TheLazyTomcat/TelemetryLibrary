type TSCSExm_TelemetryMemState = packed record

Description

Structure used when working with shared memory. Directly mapped to the MMF location.

It is binary compatible with structure declared in original SDK example, meaning you can use plugin compiled from C++ code and access the shared memory using this structure, and vice-versa.
