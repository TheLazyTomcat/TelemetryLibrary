Declaration

type TTelemetryLogBinaryReader = class(TObject)

Description

Base class for all internal readers used to write binary logs.

This class is intended as common ancestor for all internal reader classes. These classes are used internaly to read different binary log structures, and should not be used outside of parsers. All descendants must implement all abstract methods. They also may add some new methods, but they should be used in parsers wery carefully and only if really needed.

Stream from which the reader will read. Must not be Nil. Position must be set directly behind the file header and API info section, not at the beginning of the Stream itself.

Information about input file. It must be completely filled by parser that is creating current instance in order for this class to work properly. Provided information are used as parameters when creating telemetry info provider. When not filled, the behavior of this class is not defined.

procedure StartReading; virtual;

Method that must be called before any log is read. Its job is to prepare anything that is necesary for actual reading (variables initialization, ...).
Sets ReadingTerminated to False and Stream position to FirstStreamPosition.
Can be used to effectively reset the reading.

procedure EndReading; virtual;

Method that must be called before destruction of instance. Its job is to free any allocated resources.

Function ReadNextLogEntry(IncreaseLogCounter: Boolean = True): Boolean; virtual; abstract;

Reads next log entry from current position in Stream. When parameter IncreaseLogCounter is set to True and result is also True, then LogCounter is increased by one.
Read data are processed and passed to appropriate event.

Parameters
IncreaseLogCounter
Determines whether to increase log counter.
Returns

True when log was read successfuly, False otherwise.

procedure ReadAllLogEntries; virtual;

Reads all log entries from current position in Stream.
Repeatedly calls ReadNextLogEntry until it returns False.)

Function ReadLogEntry(Index: Integer): Boolean; virtual; abstract;

Reads log entry selected by passed index. Works only for structures that supports non-sequential reading (eg. structure 1).
Position in Stream is not changed in this method.

Parameters
Index
Index of requested log entry.
Returns

True when the entry was read successfuly, False otherwise.

property LogEntries[Index:Integer]: TLogEntry read GetLogEntry;

Array property mapped to log entries - that is, each item in this array corresponds to one log entry.
When you try to access item that is out of allowed boundary or the reading fails, then it returns InvalidLogEntry.
Works only for structures that supports non-sequential reading.
