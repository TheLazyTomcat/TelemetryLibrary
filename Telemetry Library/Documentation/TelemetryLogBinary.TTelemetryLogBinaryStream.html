TelemetryLogBinary

Declaration

type TTelemetryLogBinaryStream = class(TTelemetryRecipientBinder)

Description

Class designed to log all traffic on telemetry API as a binary log into provided stream.

Only file header and API informations are actually written in this class, everything else (log data) is written using instance of appropriate (for selected file structure) writer.
*Handler methods all require valid telemetry recipient object. It can be provided in Recipient property (prefered) or in Sender parameter of a method. If you fail to provide it, these methods raise an exception.

constructor Create(Recipient: TTelemetryRecipient; Stream: TStream; DataStructure: Word = 0);

Class constructor.

Creates internal log writer and starts writing.

Parameters
Recipient
Telemetry recipient that operates on API. You must provide valid instance as API informations section will be filled usig its properties. Constructor also assigns all handler methods to its event - you can change that (deassign handlers, set Recipient property to other instance, etc.) after the constructor returns, but rememeber that handlers require valid recipient instance either in Recipient property or passed in Sender paramater.
Stream
Stream to which the output will be written.
Position should be set to a beginning, but it is not mandatory.
DataStructure
Structure of output binary log. See binary log documentation for details. If you pass unsupported structure, the constructor raises an exception.
destructor Destroy; override;
