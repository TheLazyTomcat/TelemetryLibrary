Unit TelemetryLogBinaryParser

DescriptionUsesClasses, Interfaces, Objects and RecordsFunctions and ProceduresTypesConstantsVariables

Description

Contains classes designed to parse binary logs.

TelemetryLogBinaryParser

©2013-2016 František Milt, all rights reserved.

Last change: 2016-03-19

This unit contains classes that are designed to parse binary logs and/or convert them to text logs, namely: 

  TTelemetryLogBinaryReader
   |- TTelemetryLogBinaryReader_0
       |- TTelemetryLogBinaryReader_1

  TTelemetryLogBinaryStreamParser
   |- TTelemetryLogBinaryFileParser
       |- TTelemetryLogBinaryToTextConverter

Included files:

    .\Inc\TelemetryMulticastEvents.pas
      Contains declarations and implementations of multicast event classes.

Classes, Interfaces, Objects and Records

Name Description
record TLogEntry Structure returned by parser when you access specific log in the stream by its index.
Class TTelemetryLogBinaryReader Base class for all internal readers used to write binary logs.
Class TTelemetryLogBinaryReader_0 Internal reader class used to read file of structure 0.
Class TTelemetryLogBinaryReader_1 Internal reader class used to read file of structure 1.
Class TTelemetryLogBinaryStreamParser Class designed to parse a stream containing binary log.
Class TTelemetryLogBinaryFileParser Class designed to parse a file containing binary log.
Class TTelemetryLogBinaryToTextConverter Converts given binary log file to a text log file.
